Hi, I'm **Horst Herb** — a developer passionate about open source software and AI-powered tools for research.

## What I'm Working On

I'm the lead developer of **[BMLibrarian](https://github.com/hherb/BMLibrarian)**, an AI-enhanced multi-agent biomedical literature research system. The project aims to help researchers navigate and analyze scientific literature more efficiently using modern AI capabilities.

## About This Blog

This site serves as a documentation hub for my projects. Here you'll find:

- **Developer guides** — Tutorials and quickstart guides for contributors
- **Architecture notes** — Deep-dives into system design and implementation details
- **Project updates** — Release notes and roadmap discussions

## Get Involved

BMLibrarian is open source and welcomes contributions. Check out the [GitHub repository](https://github.com/hherb/BMLibrarian) to get started, or browse the posts on this blog for developer documentation.
